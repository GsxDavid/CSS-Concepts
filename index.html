CSS Layout

SS Layout se refiere a cómo se estructuran y colocan los elementos en una página web utilizando estilos CSS. En otras palabras, el diseño CSS se ocupa de la disposición y el posicionamiento de los elementos en la interfaz de usuario. Un buen diseño ayuda a organizar la información de manera clara y atractiva, facilitando la comprensión y navegación para los usuarios.

Algunos aspectos clave del CSS Layout incluyen:

  • Position
  • Flexbox
  • Grid

Position

Flexbox

Flexbox, o "Flexible Box", es un modelo de diseño en CSS que proporciona una manera eficiente y predecible de distribuir espacio entre elementos y alinearlos de manera flexible, independientemente de su tamaño y orden original. Con Flexbox, puedes crear diseños complejos y realizar alineaciones de manera más sencilla que con los modelos de diseño tradicionales.

Propiedades

Es importante entender cómo funcionan las siguientes propiedades para trabajar correctamente con flexbox

Justify content (justify-content)

Centra los elementos a lo largo del eje principal, ya sea horizontal o verticalmente, según la dirección del contenedor (row o column).

Align Items (align-items)

Alinear los elementos a lo largo del eje cruzado

Flex wrap

flex-wrap es una propiedad de CSS que se utiliza en contenedores flexibles (display: flex;) para especificar si los elementos hijos deben envolverse o no dentro del contenedor flexible. En otras palabras, controla cómo se manejan los elementos cuando no hay suficiente espacio en el contenedor principal.

  • nowrap: Los elementos flexibles se colocarán en una sola línea horizontal sin envolver. Si no hay suficiente espacio, los elementos pueden desbordar del contenedor.
  • wrap: Los elementos se envuelven a la siguiente línea si no hay suficiente espacio en la fila actual. Se crean nuevas filas según sea necesario.
  • wrap-reverse: Similar a wrap, pero las nuevas filas se crearán en sentido inverso, de abajo hacia arriba.

Flex Grow

Es una propiedad que determina la capacidad de un elemento flexible para crecer en relación con otros elementos flexibles dentro del mismo contenedor cuando hay espacio adicional disponible en el eje principal.

Flex Shrink

flex-shrink es una propiedad que determina la capacidad de un elemento flexible para contraerse en relación con otros elementos flexibles dentro del mismo contenedor cuando hay escasez de espacio en el eje principal.

Grid

El concepto de Grid (o cuadrícula) en diseño web se refiere a un sistema de diseño bidimensional que permite organizar el contenido en filas y columnas. El diseño basado en cuadrícula ofrece una estructura flexible y poderosa que facilita la creación de diseños complejos y alineados.
